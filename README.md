# Google_Playstore_Dashboard
A data analysis and visualization project using Excel and Power BI based on the Google Play Store dataset.

# Tools Used
Microsoft Excel
Microsoft Power BI

# Project Features
Data cleaning in Excel
Interactive Power BI dashboard
KPI cards and charts
App category analysis
Free vs Paid app comparison
Ratings and installs analysis

# Dashboard Insights
Most apps are free
Family and Game categories dominate
Higher reviews usually mean higher installs
Most apps have ratings above 4.0
